## To-Do List

A simple to-do list app built with React Router v7, React 19, amd Tailwind CSS v4.

## Features 

- Add, complete, and delete tasks
- Tasks persist in localStorage
- Keyboard support (Enter to add a task)

## Getting Started

### Prerequisites

- Node.js 20 or higher

### Install

```sh
npm install
```

### Development

```sh
npm run dev
```

### Production Build 

```sh
npm run build
npm start
```

### Docker

```sh
docker build -t to-do-list .
docker run -p 3000:3000 to-do-list
```

## Tech Stack

- React 19
- React Router 7
- Tailwind CSS 4
- Vite 8
- TypeScript
